
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation / Build

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
